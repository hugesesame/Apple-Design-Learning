Module 11 · コンテンツ

コンテンツ設計

ビジュアルは感情を作り、言葉が意味を作る。Appleのコピーは視覚デザインと同等の精度で設計されている——その言語パターンを解剖する。

コンテンツはデザインの一部だ

多くのデザインプロセスでは「仮テキスト（Lorem ipsum）を入れておいて後でコピーライターが書く」というワークフローをとる。Appleはこれをしない。

apple.comのすべてのコピーはデザインと同時に、時にデザインより先に設計される。ヒーロータイトルが7文字か12文字かで、フォントサイズが変わる。フォントサイズが変わると、余白が変わる。余白が変わると、製品ビジュアルの位置が変わる——つまりコピーの決定はレイアウトの決定だ。

コンテンツは容器に流し込むものではない。
コンテンツがデザインの骨格だ。

Apple Editorial Philosophy

ボイスとトーン

Appleのブランドボイスは変化しない。トーンがコンテキストで変わる。

属性意味例
Direct（率直） 回りくどくない。余分な修飾語を使わない。 「最も薄い。最も軽い。」← 主語を省く大胆さ。
Confident（自信） 疑問形を使わない。断言する。 「あなたの仕事を、変える。」（「〜できるかも」は使わない）
Human（人間的） 技術用語を平易な言葉で言い換える。 「A18 Pro」ではなく「飛躍的な進化」
Aspirational（高揚感） ユーザーが「なりたい自分」に語りかける。 「あなたのクリエイティビティを、解放する。」
Elegant（上品） 感嘆符・過激な表現・スラングを使わない。 「凄い！」ではなく「驚異的な。」

ヒーローコピーの構造

apple.comのヒーロータイルはタイトル + サブタイトルの2層構造だ。タイトルで感情を動かし、サブタイトルで具体的な事実を伝える。

誤りのパターン
最新のスマートフォン。M3チップ搭載、容量最大512GB、カメラ4800万画素。
Appleのパターン
iPhone 16 Pro。
プロが求めるすべて。
誤りのパターン
当社の高性能ノートパソコンを使って、あなたのビジネスをもっと効率化しませんか？
Appleのパターン
MacBook Air。
無敵のパフォーマンス。
ゼロの妥協。
誤りのパターン
私たちは素晴らしい製品を作るために日々努力しています。ぜひご覧ください！
Appleのパターン
Think different.

短縮の技法

Appleのタイトルは多くの場合、主語を省く。「それは最も薄い」ではなく「最も薄い。」。主語のない断言は読者自身が主語を補い、より強い感情を生む。

CTAの言葉

CTAボタンのテキストはapple.comで数種類に絞られている。「今すぐ」「クリック」などの煽り文句は存在しない。

日本語英語用途
詳しく見る Learn more 製品詳細ページへ。最も多用されるCTA。
購入する Buy 購入フローの起点。主CTAの定番。
比較する Compare 比較ページへ。セカンダリCTAとして使われる。
すべて見る See all 一覧ページへのリンク。コレクション文脈で使用。
詳しく見る Learn more → インラインテキストリンク形式（chevronなし/あり）

禁止ワード

「今すぐ」「クリック」「急いで」「限定」「チェック」— これらの煽動的・マーケティング的な言葉はAppleのCTAに存在しない。CTAはユーザーを急がせるのではなく、次のステップへ静かに招待する。

数字の表現

Appleは数字を「情報」として伝えるのではなく、「驚き」として演出する。技術仕様の羅列ではなく、一つの数字を大きく見せることで記憶に残す。

4K
解像度をただ「4K」と書かずに大きな数字として見せる
18時間
バッテリー駆動時間を断言値として提示
2×
前世代比の倍数表現（具体的な数値より力強い）

数字の文法

  • 数字は算用数字を使う（「十八時間」ではなく「18時間」）。
  • 単位は数字の直後に置く（「18 時間」ではなく「18時間」）。
  • 大きな数字にはKやmを使う（「3,000,000」ではなく「3M」）。
  • 比較には「前世代比」「競合比」を明記する。根拠を示す。

製品名の使い方

Appleの製品名はシステムとして設計されており、使い方にルールがある。

ファミリー名のみ（= 大文字始まり）
iPhone、MacBook、iPad、AirPods — 単独で使える。冠詞「a / the」不要。
モデル名（= ファミリー + サフィックス）
iPhone 16 Pro、MacBook Pro 16インチ — スペース区切り。「M4 チップ搭載 MacBook Pro」
チップ名（= 大文字 + 数字）
M4、A18 Pro — 必ずこの形式。「M4チップ」は正しい。「M4 チップス」は誤り。
禁止パターン
「i Phone」（スペース不可）/ 「iphone」（小文字不可）/ 「Apple iPhone」（ブランド名の重複）

テキスト量と視覚的バランス

Appleのコピーは短い。しかしその「短さ」は偶然ではなく、視覚的な余白を最大化するための設計上の選択だ。

Pro. そのひとことに、すべてが宿る。
MacBook Pro
詳しく見る 購入する
テキスト少 → 余白多 → 製品が主役
この革新的な最新テクノロジーを搭載した私たちの最高傑作。業界最高水準のパフォーマンスをお届けします。
あなたのビジネスとクリエイティビティを次のレベルへ引き上げるために設計された一台です。
今すぐ詳細を確認する 比較して購入を検討する
テキスト多 → 余白消える → ごちゃごちゃ

製品ページのヒーロータイトルは基本的に全角20字以内（英語で5〜8単語）を目指す。それ以上になる場合は2行に分けるか、コピーを削る。

演習

Appleスタイルのコピーを書く

あなたが関わるプロダクト（自分のポートフォリオ、個人サービス、または仮想の製品）のapple.comスタイルヒーローコピーを書いてください。

  1. 製品の「最も重要な1つのベネフィット」を選ぶ。機能ではなく、ユーザーの体験として表現する（例: 「高速SSD搭載」→「待つ時間が、消える。」）。
  2. タイトルを全角20字以内（英語なら8単語以内）で書く。主語を省き、断言する。
  3. サブタイトルを1〜2文で書く。ここで初めて「何ができるか」を具体的に伝える。
  4. CTAを「詳しく見る」「購入する」「試す」の3語以内で選ぶ。煽りワード禁止。
  5. 書いたコピーをFigmaのapple.comテンプレートに配置し、スクリーンショットをコミュニティに投稿する。「どのボイス属性（Direct / Confident / Human など）を意識したか」を1行で添える。
